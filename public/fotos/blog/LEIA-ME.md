# Fotos do blog

Use esta pasta para as capas dos artigos do blog.

## Regra simples

- Formato ideal: JPG.
- Proporcao ideal: 16:9.
- Tamanho recomendado: 1600x900 px.
- Tamanho minimo aceitavel: 1200x675 px.
- Peso recomendado: ate 500 KB quando possivel.

A mesma foto aparece em dois lugares:

- Aba Blog: card de previa do artigo.
- Pagina do artigo: imagem grande logo abaixo do titulo.

Como os dois lugares usam a mesma imagem, mantenha o assunto principal no centro da foto. O site corta com `object-fit: cover` para preservar o acabamento.

## Como trocar sem quebrar

1. Exporte a nova foto em JPG, 1600x900 px.
2. Renomeie exatamente com o mesmo nome do arquivo atual.
3. Substitua o arquivo nesta pasta.
4. Atualize/recarregue o site.

Se quiser usar outro nome, altere o campo `image` no arquivo Markdown correspondente em `src/content/blog`.
