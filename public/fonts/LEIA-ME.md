# Pasta de Fontes — Crystal Lashes Site

## Fonte necessária

| Fonte | Arquivo esperado | Uso no site |
|---|---|---|
| Better Signature Regular | `better-signature-regular.woff2` | "Crystal Lashes" no Header e assinaturas pontuais |

## Instrução

1. Coloque o arquivo da fonte aqui nesta pasta com o nome exato: `better-signature-regular.woff2`
2. Se o arquivo que você tem for `.otf` ou `.ttf`, avise o Rodrigo para converter para `.woff2` antes de usar (conversão via fontsquirrel.com ou cloudconvert.com)
3. Assim que o arquivo estiver aqui, avise para configurar o `@font-face` no CSS

## Por que WOFF2?

WOFF2 é o formato moderno de fonte para web. É até 30% menor que WOFF e tem suporte em todos os navegadores atuais. Garante carregamento mais rápido sem perda de qualidade visual.
